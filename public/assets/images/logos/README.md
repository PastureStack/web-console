# PastureStack Brand Assets

This directory contains PastureStack-owned interface marks plus inherited non-brand illustrations that remain subject to the repository license and history.

Active PastureStack assets:

- `pasturestack-mark.svg` — primary full-color mark
- `pasturestack-mark-light.svg` — mark for dark navigation surfaces
- `pasturestack-wordmark.svg` — about-screen wordmark
- `pasturestack-favicon.svg` — browser favicon
- `pasturestack-welcome-scene.svg` — welcome illustration
- `pasturestack-failure-node.svg` — error-screen recovery illustration

Do not restore removed upstream product logos, animal marks, or product-specific scene artwork. Compatibility API names do not authorize reuse of upstream branding.
